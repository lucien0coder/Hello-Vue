#功能说明
  * 账号输入框
   * 1、正则表达式匹配账号名称是否符合要求
   * 2、输入过程中自动匹配数据，验证账号是否有重复
   * 3、提示正在验证状态、小字体提示是否符合规范或者不合规范
  * 初次输入密码
   * 无特殊要求 
  * 确认密码
   * 1、与初次输入密码进行匹配
   * 2、提示密码一致或者不一致(focus后提示重新输入)
  * checkbox项
   * 1、链接打开展示“用户协议” 
   * 2、勾选后，确认按钮状态为able
  * 确定按钮
   * 1、默认为disable状态
   * 2、点击确认后，提交数据
  * 返回按钮
   * 返回某页面
